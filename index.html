About

Hey there! I'm an aspiring game developer from Iceland, passionate about creating and designing everything from code to art. I began my journey into game development at the age of 12, and video games have always played a significant role in my life.

I have completed a year in an MSc Software Engineering program, I possess a strong background in programming, problem-solving, and game design. My professional experience includes working as a machine learning researcher and as a software developer.

On this site, you'll find some of my game development projects. Currently, I'm concentrating on a full-fledged tactical roguelike title. With the technical aspects nearly complete and a promising prototype in hand, I'm shifting my focus to developing assets and marketing strategies to craft a game I can be truly proud of.

Personal Projects

Unannounced Rougelite Tactics Game

Independently developing a tactical turn-based game in Godot, focusing on recruitment and sacrifices, combining elements from popular indie games such as Darkest Dungeon, Into the Breach, and Slay the Spire.

    Highlights:

  • Optimised AI systems designed for handling large feature space
  • Readable AI Targeting system for skill-based gameplay
  • Robust Class Tree system implementation
  • Randomly generated events for dynamic gameplay

2.5D Animation System

Explore the potential of rigged vector animation in game engines for expressive and dynamic 2.5D animations. This technology allows for in-game modifiers, enabling thousands of expressions with smoother animations with stylised look.

    Highlights:

  • Converts .SVG files into custom lines and polygon gameobjects
  • Capability to handle complex rigs and transformations
  • Features timeline, keyframes, curves, and sliders for animation
  • Offers load and save system
  • Designed for the Godot game engine

School Projects

VR Phobia Based Horror

Immersive VR horror game where fear is intensified by targeting phobias without relying on jumpscares or deadly situations. Players face uncomfortable scenarios, such as walking near cliff edges among enormous spiders.

    Highlights:

  • Teacher's feedback: "That was bloody incredible"
  • 15 sections with scripted events and animations
  • VR movement system with collision prevention
  • Phone feature: vibration, screen, and flashlight
  • Underwater effects
  • Developed using the Unity Game Engine

Hellevator

A 3-week full-time group project where we created a 3D top-down action game. The game features Jesus Christ descending to hell via an elevator to rescue his friend Greg.

    My contributions included:

  • AI mechanics
  • Ranged unit implementation and modelling
  • Projectile and deflection mechanics
  • Player upgrade system
  • Developed using the Unity Game Engine

The Battle for Midjord

Story based project that emphasises on effective storytelling by putting the player through difficult moral choices. The game features three possible endings based on the player's decisions.

    Highlights:

  • Fantasy world where humans advance through new technologies
  • Animated characters brought to life with modern AI voice acting
  • Interactive object pickup system
  • Developed using the Unity Game Engine

Graphical Programing using OpenGL
